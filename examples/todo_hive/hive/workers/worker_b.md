# Worker B 任務分配

## 任務概述

Worker B 負責探索以下功能領域的實現方案：

---

## 任務 1：User Registration（用戶註冊）

### 功能需求

根據 README.md 的規範：
- User registration with username and password（用戶註冊，包含用戶名和密碼）
- 無授權角色
- 無密碼重置
- 無第三方身份提供者

### 探索方向

Worker B 可以探索以下實現方案：

#### 認證存儲選項
- **方案 B1**：檔案基礎用戶存儲（file-based user storage）
  - 使用 JSON 檔案存儲用戶資料
  - 提供持久化能力
  - 可讀寫用戶資料

#### 密碼處理
- 密碼哈希策略（bcrypt、scrypt、PBKDF2）
- 密碼驗證流程
- 錯誤處理（用戶名已存在、無效輸入等）

#### 用戶標識符
- UUID 作為用戶標識符
- 用戶名作為唯一標識符
- ID 生成策略

### 實現要求

- Python 基礎實現
- HTTP 基礎 API
- 明確分離認證領域
- 使用 JSON 檔案持久化
- 使用 UUID 作為標識符（如適用）

## 探索原則

作為 Worker，Worker B 的職責是：

1. **探索替代實現**：可以實現重疊功能的不同方案
2. **無需決策責任**：不負責最終決策
3. **獨立探索**：可以與 Worker A 探索不同的路徑
4. **記錄探索過程**：記錄實現選擇和理由
